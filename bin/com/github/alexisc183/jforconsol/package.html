Provides classes and interfaces mainly for CLI app development.

The classes and interfaces of this package can be classified as follows:
  • Aid over 'missing' features of the language
  • Aid over 'missing' built-ins of the Java(TM) SE platform
  • Safety on input requests
  • Friendly output visualization on CLI environments
  • Other
The purpose of this package is to speed up the coding process of programs that may be run solely for debugging purposes or applications that make use of a command-line interface (CLI).
